# 闭包
闭包概念："有权访问另一个函数作用域中变量的函数"

闭包的最大作用是“隐藏变量”，一大特性是**内部函数总是可以访问外部函数中声明的参数和变量，甚至在其外部函数被返回（寿终正寝）之后**，比如可以利用闭包访问私有变量。

缺点也很明显：

* 常驻内存，增加内存使用量

* 使用不当造成内存泄漏（IE中的情况）

例子：
```javascript
function fun(){
  var num = 1;
  return function () {
    console.log(++num)
  }
}

var count = fun()
count()//2
count()//3
count()//4
```
