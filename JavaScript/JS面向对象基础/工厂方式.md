### 工厂方式
* 工厂方式
    * 用构造函数创建一个类
    * 什么是类、对象（实例）：模具和零件
    * 笔记：
        * 构造对象的函数
        * ` constructor ` 属性 返回所有 ` JavaScript ` 变量的构造器函数
* 工厂方式的问题
    * 没有 new
    * 函数重复定义
* 问题解决：**构造函数加上** ` new `，**然后用原型** ` Prototype ` 为对象添加**方法**
    * ` new ` 做了两件事
        * **替你创建了一个空白对象**：` var this = new Object() `
        * **替你返回了这个空白对象**： ` return this `
        * ` new ` 和 ` this `
