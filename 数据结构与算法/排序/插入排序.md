# 插入排序

```javascript
function insertionSort (arr) {
    let current
    let len = arr.length
    for (let i = 0;i < len;i++) {
        current = arr[i]
        while (i>=0 && arr[i-1] > current) {
            arr[i] = arr[i-1]
            i--
        }
        arr[i] = current
    }
    return arr
}
```
