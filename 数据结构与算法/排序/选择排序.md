# 选择排序
1. 在未排序序列中找到最小（大）元素，存放到排序序列的起始位置
2. 从剩余未排序元素中继续寻找最小（大）元素，然后放到已排序序列的末尾
3. 重复第二步，直到所有元素均排序完毕
```javascript
const selectionSort = function (arr) {
    let len = arr.length
    let minIndex, temp
    for (let i = 0;i < len - 1;i++) {
        minIndex = i
        for (let j = i + 1;j < len;j++) {
            // 寻找最小的数
            if (arr[j] < arr[minIndex]) {
                // 将最小数的索引保存
                minIndex = j
            }
        }
        temp = arr[i]
        arr[i] = arr[minIndex]
        arr[minIndex] = temp
    }
    return arr
}
```
