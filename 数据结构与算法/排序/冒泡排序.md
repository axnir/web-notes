# 冒泡排序

```javascript
function bubbleSort (arr) {
    let temp
    for (let n = arr.length; n >= 1;n--){
        for (let i = 0;i < n-1;i++){
            if (arr[i] > arr[i+1]) {
                temp = arr[i]
                arr[i] = arr[i+1]
                arr[i+1] = temp
            }
        }
    }
    return arr
}
```
