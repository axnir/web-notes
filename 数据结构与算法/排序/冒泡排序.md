# 冒泡排序

```javascript
function bubbleSort (arr) {
    let len = arr.length
	// 不必到最后一个元素，最后一个元素后边没有元素了
    for (let i = 0; i < len-1; i++) {
        // 最后一个就是最大的，下一次不需要再比较
        for (let j = 0; j< len - i - 1; j++) {
            if(arr[j] > arr[j+1]) {
                [arr[j], arr[j+1]] = [arr[j+1], arr[j]]
            }
        }
    }
    return arr
}
```
